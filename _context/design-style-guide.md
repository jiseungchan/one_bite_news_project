# 병규의 한입뉴스 — 디자인 스타일 가이드

> Heisenberg 디자인 분석을 기반으로 한 한입뉴스 전용 가이드

---

## Heisenberg 디자인 분석

스크린샷에서 추출한 Heisenberg의 핵심 디자인 패턴:

| 요소 | Heisenberg 구현 방식 |
|------|---------------------|
| 헤더 | 다크 네이비(거의 블랙) + 흰색 로고 + 수평 내비게이션 |
| 레이아웃 | 3~4열 카드 그리드, 깔끔한 여백 |
| 카드 | 16:9 썸네일 + 카테고리 태그 + 굵은 제목 + 짧은 설명 |
| 타이포 | 산세리프, 제목은 크고 굵게, 본문은 작고 읽기 좋게 |
| 색상 | 다크 헤더 / 흰 본문 배경 / 강조색 최소 사용 |
| 아티클 | 와이드 레이아웃, 큰 히어로 이미지, 긴 세로 스크롤 |
| 카테고리 구분 | 섹션 헤더로 카테고리 묶음 표시 |

**한입뉴스는 이 구조를 그대로 가져오되, 한국어와 병규의 감성에 맞게 조정한다.**

## 컬러 시스템

Heisenberg의 다크 헤더 + 화이트 본문 구조를 그대로 채택.

```
헤더 배경
  #111827  (Heisenberg의 다크 네이비와 유사한 거의 블랙에 가까운 네이비)

본문 배경
  #FFFFFF  (흰색)
  #F9FAFB  (섹션 구분용 옅은 그레이)

텍스트
  #111827  (제목 — 헤더 색과 동일, 강한 대비)
  #374151  (본문)
  #6B7280  (날짜, 출처, 메타 정보)
  #FFFFFF  (헤더 위 텍스트)

강조색 (링크, 태그 등)
  #2563EB  (블루 — Heisenberg 계열)
  호버: #1D4ED8

카테고리 태그 색상
  AI:       #7C3AED  (바이올렛)
  바이오:   #059669  (에메랄드)
  나노·소재: #D97706  (앰버)
  물리·우주: #0284C7  (스카이블루)
  에너지:   #16A34A  (그린)
  경제·주식: #0F766E  (틸)
  이번 주 픽: #DC2626  (레드)
```

## 타이포그래피

### 폰트

```
한글 본문 및 제목
  Pretendard (권장) — 가장 현대적이고 Heisenberg 느낌에 가까움
  대안: Noto Sans KR — 안정적, 무료

영문·숫자·카테고리 레이블
  Inter — Pretendard와 잘 어울림

둘 다 Google Fonts / CDN으로 무료 사용 가능
```

### 타입 스케일

| 역할 | 크기 | 굵기 | 용도 |
|------|------|------|------|
| Display | 40px | 800 | 메인 히어로 제목 |
| H1 | 30px | 700 | 아티클 제목 |
| H2 | 22px | 700 | 섹션 제목 |
| H3 | 18px | 600 | 카드 제목 (2줄 이내) |
| Body | 16px | 400 | 본문 |
| Small | 14px | 400 | 카드 설명, 부연 |
| Caption | 12px | 400 | 날짜, 출처 |
| Tag | 11px | 600 | 카테고리 레이블 (대문자) |

### 행간
- 제목류: `line-height: 1.3`
- 본문: `line-height: 1.75`  (한국어 가독성 위해 여유 있게)

## 컴포넌트 가이드

### 뉴스 카드

Heisenberg 카드 구조를 그대로 사용:

```
┌────────────────────────┐
│  [썸네일 이미지 16:9]   │
├────────────────────────┤
│  [AI] 카테고리 태그     │
│                        │
│  제목 텍스트            │
│  (H3, 최대 2줄 말줌임)  │
│                        │
│  요약 1~2줄 (Small)     │
│                        │
│  2024.01.15 · 출처명   │
└────────────────────────┘
```

**인터랙션**
- 호버: `box-shadow` 강화 + 제목 색상 → `#2563EB`
- 썸네일 호버: `transform: scale(1.03)`, `transition: 0.2s ease`
- 카드 전체 클릭 가능 (`cursor: pointer`)

### 카테고리 태그

```css
display: inline-block;
padding: 3px 8px;
border-radius: 4px;
font-size: 11px;
font-weight: 600;
letter-spacing: 0.05em;
text-transform: uppercase;
background: {카테고리색 10% 투명도};
color: {카테고리색};
```

### 헤더 내비게이션 (Heisenberg 스타일 직접 채택)

```css
background: #111827;
height: 60px;
color: #FFFFFF;

/* 활성 메뉴 */
border-bottom: 2px solid #FFFFFF;

/* 로고 */
font-size: 20px;
font-weight: 700;
color: #FFFFFF;
```

### 섹션 헤더 (카테고리 구분)

Heisenberg처럼 카테고리별로 섹션을 나눌 때:

```
[AI]  ─────────────────────────────  더보기 →
```

```css
display: flex;
align-items: center;
border-bottom: 2px solid #111827;
font-size: 18px;
font-weight: 700;
margin-bottom: 24px;
```

## 아티클 페이지

Heisenberg 아티클의 넓은 여백, 큰 이미지 구조 채택:

```
최대 너비: 720px (본문)
히어로 이미지: 풀 와이드 (aspect-ratio: 16/9)
본문 폰트: 16px, line-height: 1.75
단락 간격: 1.5em

상단에 표시:
  카테고리 태그 + 제목(H1) + 날짜 + 출처 링크
```

## 카드뉴스 슬라이드 규격

웹 외에 카드뉴스 형식으로 제작할 때:

```
정방형:  1080 × 1080px  (인스타그램 피드)
세로형:  1080 × 1920px  (스토리, 릴스)
```

### 슬라이드 구성 (표준 6장)

| # | 내용 |
|---|------|
| 1 | 표지 — 제목 + 날짜 + 로고 |
| 2 | 한 줄 요약 (이게 왜 흥미로운가) |
| 3~5 | 핵심 포인트 1개씩 |
| 6 | 출처 + 원문 보기 + 로고 |

### 슬라이드 타이포

```
표지 제목:    48px / 800 / 흰색 (다크 배경)
포인트 제목:  32px / 700
설명 텍스트:  20px / 400
캡션:        14px / 400 / 그레이
```
