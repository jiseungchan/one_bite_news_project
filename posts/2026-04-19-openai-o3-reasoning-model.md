## 한 줄 요약
> OpenAI o3는 단순 답변 생성이 아니라 '풀이 과정을 스스로 검토하는' 구조로 설계돼, 수학 올림피아드 수준의 문제를 풀 수 있는 첫 번째 상업용 AI 모델이 됐다.

## 리드 — 왜 지금 이게 중요한가?

수학 올림피아드 예선 문제 25개 중 25개를 맞혔다. 2024년 말 o1이 같은 테스트에서 9개를 맞힌 것과 비교하면 격세지감이다. OpenAI가 공개한 새 추론 모델 o3는 ARC-AGI(인간 수준 일반 지능을 측정하는 벤치마크) 점수에서 87.5%를 기록하며, 인간 평균(85%)을 처음으로 넘어섰다.

## 배경 — 기존에는 어땠나?

GPT 계열 모델들은 '다음 토큰을 예측하는' 방식으로 작동한다. 이 구조는 유창한 문장을 만들어내는 데는 탁월하지만, 복잡한 수학 증명이나 다단계 논리 추론에서는 한계가 명확했다. 실제로 GPT-4는 간단해 보이는 수학 문제에서도 자신 있게 틀린 답을 내놓는 '환각(hallucination)' 현상이 잦았다.

2024년 OpenAI가 공개한 o1은 이 문제를 다른 방식으로 접근했다. 답을 바로 출력하는 대신, 내부적으로 '생각하는 시간(thinking tokens)'을 가지고 스스로 추론 과정을 검토한 뒤 답을 내는 구조다. 마치 시험을 볼 때 답을 쓰기 전에 풀이 과정을 검산하는 것처럼. 이 접근법은 수학·코딩에서 눈에 띄는 성능 향상을 보여줬지만, 추론 시간이 길어지고 비용이 높다는 단점이 있었다.

## 핵심 — 무엇이 다른가?

o3는 o1의 구조를 유지하면서 두 가지를 크게 개선했다.

첫째, **추론 깊이의 조절이 가능**해졌다. 사용자가 '빠른 추론'과 '깊은 추론' 중 선택할 수 있다. 간단한 질문에는 빠르게, 복잡한 수학 증명에는 더 오래 생각하도록 설정하는 식이다. 이는 추론 비용을 태스크에 맞게 최적화할 수 있다는 뜻이다.

둘째, **외부 도구 검색을 추론 과정에 통합**했다. o3는 풀이 도중 웹 검색이나 코드 실행이 필요하다고 판단하면 스스로 도구를 호출한다. 기존 모델이 '검색 → 답변'의 선형 구조였다면, o3는 '추론 중 검색 → 추론 재개 → 답변' 방식으로 동작한다.

| 항목 | GPT-4o | o1 | o3 |
|------|--------|----|----|
| 추론 방식 | 즉시 생성 | 내부 검토 후 생성 | 조절 가능한 심층 추론 |
| 도구 통합 | 별도 단계 | 제한적 | 추론 중 자동 통합 |
| ARC-AGI 점수 | 약 5% | 32% | 87.5% |
| 수학 올림피아드 | — | 9/25 | 25/25 |

## 시장 영향 — 어디에 쓰이나?

- 법률 문서 검토: 복잡한 계약서에서 상충 조항을 단계별로 추론해 식별
- 소프트웨어 엔지니어링: SWE-bench(실제 Github 이슈 해결 능력 테스트)에서 71.7% 기록 — 전문 개발자 수준
- 의료 진단 보조: GPQA(대학원 수준 과학 문제 벤치마크) 점수 87.7%, 전문가 패널 평균 초과
- API 가격: o1 대비 약 2배 수준이나, 추론 효율이 높아 문제당 실질 비용은 유사하거나 낮을 수 있다는 분석

## 병규의 한 줄

이번에 흥미로운 건 수치보다 구조의 변화다. '더 많이 학습'이 아니라 '더 오래 생각'하는 방향으로 성능을 끌어올린 것인데, 이는 AI 확장의 무게중심이 파라미터 수에서 추론 시간으로 이동하고 있다는 신호다.
