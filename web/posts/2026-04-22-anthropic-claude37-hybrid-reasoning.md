이번에 흥미로운 건 모델의 '생각하는 방식'을 사용자가 직접 제어할 수 있게 됐다는 점이다. 앤트로픽이 공개한 클로드 3.7 소네트는 같은 모델이 상황에 따라 빠른 반사 반응을 할 수도, 느리고 깊은 추론을 할 수도 있다.

## 배경 — 하이브리드 추론이 왜 필요한가

AI 모델에는 오래된 딜레마가 있다. 빠르게 답하면 복잡한 문제에서 틀리고, 깊이 생각하면 간단한 질문에도 수십 초가 걸린다. OpenAI의 o1·o3 시리즈가 "더 오래 생각할수록 더 잘 푼다"는 CoT(Chain-of-Thought — 단계별 사고 과정을 거쳐 추론하는 기법)의 가능성을 입증했지만, 매번 긴 추론을 돌리면 비용과 응답 시간이 치솟는다.

"간단한 질문엔 바로 답하고, 수학 증명이나 복잡한 코딩엔 오래 생각하게 하면 안 되나?" — 이 당연해 보이는 요구가 실제 구현에선 상당히 어렵다. 같은 모델이 두 가지 추론 방식을 모두 잘 수행해야 하기 때문이다.

## 핵심 — 생각 예산이란 무엇인가

클로드 3.7 소네트의 핵심 기능은 '생각 예산(Extended Thinking Budget)'이다. API 호출 시 `thinking` 파라미터로 최대 사고 토큰 수를 지정할 수 있다. 1,024개로 설정하면 간결한 추론, 10,000개 이상으로 설정하면 복잡한 문제를 단계별로 분해해 풀어나간다.

| 모드 | 특징 | 적합한 상황 |
|------|------|-----------|
| 일반 모드 | 즉각 응답, 낮은 비용 | 간단한 Q&A, 번역, 요약 |
| 확장 추론 | 단계별 사고, 높은 정확도 | 수학 문제, 복잡한 코딩, 논리 분석 |

앤트로픽이 공개한 벤치마크 결과에서 클로드 3.7 소네트는 SWE-bench Verified(소프트웨어 엔지니어링 실제 작업을 평가하는 벤치마크)에서 62.3%를 기록했다. 비교 대상인 OpenAI o1은 48.9%다. 수학 올림피아드 수준 문제를 다루는 AIME 2025에서도 기존 클로드 3.5 소네트 대비 두 배 이상 향상된 성능을 보였다.

## 시장 영향 — 개발자 도구 시장의 판이 바뀐다

- API 가격은 입력 토큰 기준 백만 개당 3달러, 출력은 15달러로 책정됐다 — 기존 클로드 3.5 소네트와 동일 수준
- 확장 추론 사용 시 추가 사고 토큰에 대해 별도 과금되는 구조
- 깃허브 코파일럿, 커서(Cursor) 등 AI 코딩 도구들이 클로드 3.7 소네트 통합을 검토 중이라는 보도가 나오고 있다
- 앤트로픽 공식 발표에서 클로드 닷에이아이(Claude.ai) 프로 플랜 사용자에게 우선 제공된다고 명시했다

## 병규의 한 줄

이번 업데이트에서 주목할 건 성능 수치보다 '사용자가 모델의 추론 깊이를 직접 조절한다'는 설계 철학이다. AI를 블랙박스로 쓰는 게 아니라, 내부 연산 자원을 사용자가 협상하는 방식으로 — 이게 다음 세대 AI 인터페이스의 방향을 보여주는 것 같다.
