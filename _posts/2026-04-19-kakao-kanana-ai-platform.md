## 한 줄 요약
> 카카오 카나나는 카카오톡 안에서 AI가 '대화 상대'가 아닌 '생활 비서'로 작동하도록 설계됐다. 한국 최대 메신저 플랫폼이 AI의 실행 환경이 되는 첫 번째 실험이다.

## 리드 — 왜 지금 이게 중요한가?

월간 활성 사용자 4,800만 명. 카카오톡이 가진 숫자다. 카카오가 이 플랫폼에 AI를 얹기로 했다. 4월 19일부터 전면 오픈된 '카나나'는 카카오톡 내 별도 탭으로 접근하며, 대화 기반으로 일정 추가, 카카오페이 송금 요청, 카카오쇼핑 상품 검색, 카카오맵 길찾기까지 처리한다. 이용자가 앱을 전환하지 않고 대화만으로 카카오 생태계 대부분의 서비스를 쓸 수 있게 된 것이다.

## 배경 — 기존에는 어땠나?

카카오는 AI 분야에서 국내 빅테크 중 가장 늦게 움직인 축에 속한다. 네이버가 2021년 HyperCLOVA를 선보이고, 삼성이 가우스 시리즈를 갤럭시에 탑재하는 동안, 카카오는 챗봇 서비스 '카카오 i'를 카카오톡 채널에 부분 적용하는 수준에 머물렀다. 독자 대형 언어 모델 개발보다 외부 모델(OpenAI API 등) 위에 서비스를 얹는 방식을 택했다.

이 선택의 배경에는 카카오 특유의 리소스 배분 전략이 있었다. 자체 모델 개발에 수천억 원을 쏟기보다, 기존 카카오톡·카카오페이·카카오쇼핑 등 8,000만 계정이 연결된 생태계 자체를 AI의 실행 환경으로 만드는 데 집중하기로 한 것이다.

## 핵심 — 무엇이 다른가?

카나나의 핵심은 '생태계 연동 깊이'다. 단순히 질문에 답하는 챗봇이 아니라, 카카오 서비스 API를 직접 호출하는 에이전트 구조로 동작한다.

예를 들어 "내일 오후 2시에 치과 예약 잡고 엄마한테 알림 보내줘"라고 입력하면:
1. 카카오캘린더에 일정 추가
2. 사용자가 지정한 '엄마' 연락처로 카카오톡 메시지 발송
3. 위치 기반 근처 치과 목록을 카카오맵에서 불러와 제안

이 과정에서 별도 앱 전환 없이 카카오톡 대화창 안에서 모두 처리된다.

| 항목 | 기존 카카오 챗봇 | 카나나 |
|------|--------------|-------|
| 서비스 연동 | 개별 채널 내 제한 | 카카오 생태계 전체 |
| 동작 방식 | 키워드 응답 | 자연어 → 실제 API 실행 |
| 학습 데이터 | 범용 | 카카오 사용자 행동 패턴 반영 |
| 접근 경로 | 채널 검색 | 카카오톡 메인 탭 |

## 시장 영향 — 어디에 쓰이나?

- 카카오페이 연동으로 "친구에게 3만 원 보내줘" 수준의 자연어 송금 가능
- 카카오쇼핑 내 상품 비교·추천 자동화로 커머스 전환율 향상 기대
- 카카오모빌리티와 연동 시 택시 호출까지 대화 기반 처리 예정 (2분기 업데이트)
- 기업 대상 카나나 API 개방도 예고 — 카카오톡을 기반으로 한 B2B AI 서비스 생태계 형성 가능성

## 병규의 한 줄

카카오가 택한 전략은 "모델을 만들기보다 파이프를 만든다"는 것이다. AI 성능 경쟁에서 한 발 물러서는 대신, 4,800만 명이 이미 쓰는 앱 안에서 AI가 실제로 '일하는' 경험을 먼저 만들겠다는 선택인데 — 이것이 맞는 방향인지는 6개월 후 사용 지표가 말해줄 것이다.
